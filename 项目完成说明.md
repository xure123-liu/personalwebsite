## 🚀 启动方式

### 方式一：使用启动脚本（推荐）

**Linux/Mac:**
```bash
./portfolio-start.sh
```

**Windows:**
```cmd
portfolio-start.bat
```

### 方式二：手动启动

1. 启动后端（终端1）:
```bash
cd portfolio-backend
npm install
npm start
```

2. 启动前端（终端2）:
```bash
cd portfolio-frontend
npm install
npm run dev
```
