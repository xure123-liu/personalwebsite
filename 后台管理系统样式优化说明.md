### 2. 文件上传样式统一 ✅

**要求**: 上传文件样式统一使用头像的上传按钮样式，不加黑色背景

**实现**:
- ✅ 所有文件上传输入框统一样式
- ✅ 移除黑色背景，使用白色背景
- ✅ 文件选择按钮样式：浅灰色背景（#f5f5f5）
- ✅ 按钮文字：黑色
- ✅ 悬停效果：按钮背景变深（#e0e0e0）

**样式特点**:
```css
/* 文件上传按钮容器 */
.form-group input[type="file"] {
  background-color: #ffffff; /* 白色背景 */
  border: 1px solid #ddd;
}

/* 文件选择按钮 */
.form-group input[type="file"]::file-selector-button {
  background-color: #f5f5f5; /* 浅灰色按钮 */
  color: #000000; /* 黑色文字 */
}
```
