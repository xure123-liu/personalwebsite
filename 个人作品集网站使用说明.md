# 个人作品集网站使用说明

## 项目结构

```
ux-designer-website/
├── portfolio-backend/      # 后端API服务器
├── portfolio-frontend/     # 前端网站
├── portfolio-start.sh      # Linux/Mac 启动脚本
└── portfolio-start.bat     # Windows 启动脚本
```

## 快速开始

### 方式一：使用启动脚本（推荐）

**Linux/Mac:**
```bash
chmod +x portfolio-start.sh
./portfolio-start.sh
```

**Windows:**
```cmd
portfolio-start.bat
```

### 方式二：手动启动

1. **启动后端服务器**
```bash
cd portfolio-backend
npm install
npm start
```
后端将在 `http://localhost:3002` 运行

2. **启动前端服务器**（新开一个终端）
```bash
cd portfolio-frontend
npm install
npm run dev
```
前端将在 `http://localhost:3000` 运行

## 访问地址

- **前端网站**: http://localhost:3000
- **后台管理**: http://localhost:3000/admin
- **后端API**: http://localhost:3002

## 功能说明

### 前端网站功能

1. **首页** (`/`)
   - 个人头像和简介展示
   - 个人作品展示（支持10个分类筛选）
   - 个人思考沉淀展示
   - 留言表单

2. **个人作品** (`/works`)
   - 作品列表展示
   - 分类筛选功能
   - 点击作品可跳转到作品链接

3. **个人思考沉淀** (`/thoughts`)
   - 思考内容卡片展示
   - 支持图片和链接

4. **个人图册** (`/gallery`)
   - 图册网格展示

5. **联系我** (`/contact`)
   - 展示微信二维码、QQ二维码、邮箱
   - 留言表单

### 后台管理功能

访问 `http://localhost:3000/admin` 进入后台管理系统

1. **个人信息管理**
   - 上传个人头像
   - 编辑主标题、副标题、描述
   - 编辑个人技能
   - 上传微信二维码、QQ二维码
   - 设置邮箱

2. **作品管理**
   - 添加、编辑、删除作品
   - 设置作品名称、描述、图片、链接
   - 选择作品分类（全部、直播类、AI工具类、图像编辑类、交易类、B端系统类、游戏类、社交类、电商类、其他类）

3. **思考沉淀管理**
   - 添加、编辑、删除思考内容
   - 设置思考标题、内容、图片、链接

4. **图册管理**
   - 添加、编辑、删除图册
   - 设置图册名称、描述、图片

5. **留言管理**
   - 查看所有用户留言
   - 修改留言状态（未读、已读、已回复）
   - 删除留言

6. **友情链接管理**
   - 添加、编辑、删除友情链接
   - 设置链接名称和地址

## 技术栈

### 后端
- Node.js
- Express
- SQLite3
- Multer (文件上传)

### 前端
- React 18
- Vite
- React Router
- Axios

## 数据库

使用SQLite数据库，数据库文件位于 `portfolio-backend/portfolio.db`

首次运行会自动创建数据库表和初始化数据。

## 文件上传

上传的文件保存在 `portfolio-backend/uploads/` 目录下。

## 注意事项

1. 确保端口 3000 和 3002 未被占用
2. 首次运行需要安装依赖（npm install）
3. 上传的图片文件会保存在后端服务器的 uploads 目录
4. 数据库文件会自动创建，无需手动配置

## 样式说明

网站采用深色主题设计，完全参考提供的示意图：
- 黑色背景 (#000)
- 白色文字
- 简洁现代的布局
- 响应式设计，支持移动端

Logo使用SVG格式，显示为黑色圆形背景上的白色"XR"字母。
