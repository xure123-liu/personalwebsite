# 个人作品集网站前端

## 技术栈
- React 18
- Vite
- React Router
- Axios

## 安装依赖

```bash
npm install
```

## 运行开发服务器

```bash
npm run dev
```

前端默认运行在 `http://localhost:3000`

## 构建生产版本

```bash
npm run build
```

## 页面说明

- `/` - 首页（包含所有主要模块）
- `/works` - 个人作品页面
- `/thoughts` - 个人思考沉淀页面
- `/gallery` - 个人图册页面
- `/contact` - 联系页面
- `/admin` - 后台管理系统

## 功能特性

### 前端功能
1. **首页**
   - 个人头像和简介展示
   - 个人作品展示（支持分类筛选）
   - 个人思考沉淀展示
   - 留言表单

2. **作品页面**
   - 作品列表展示
   - 分类筛选
   - 作品详情

3. **思考沉淀页面**
   - 思考内容卡片展示

4. **图册页面**
   - 图册网格展示

5. **联系页面**
   - 联系方式展示（微信二维码、QQ二维码、邮箱）
   - 留言表单

### 后台管理功能
1. **个人信息管理** - 管理头像、简介、联系方式等
2. **作品管理** - 添加、编辑、删除作品
3. **思考沉淀管理** - 管理思考内容
4. **图册管理** - 管理个人图册
5. **留言管理** - 查看和管理用户留言
6. **友情链接管理** - 管理友情链接
